Methods to apply stylings to web page
-------------------------------------
    1.Inline css/stylings     
        -> apply styles by taking 'style attribute' in html element
        ->this styling is applicable for only that element
        ->for inlinr css not required
        -> inline css is more priority than internal and external css

    2.internal css/ stylings
        ->applying styles by taking a styletag in head part is called as a internal styling or css
        ->to apply internal stylings we need css selectors 
        
    3.external css/stylings
        ->apply styling by taking new .css file and link to file by using <link></link> tag 
        ->to aply External CSS we need CSS selectors

        5.[AttributeName*="value"]: it will select all if it match any letter it will match(universal)

media queries:
----------------------------------
media queries are used to build reponsive web page based on device width(device view port)
   -> break points
   1.smaller devices (mobile with brands) 0px-450px
   2.media devices (tables/ipad) 450px - 850px
   3.large devices (laptops ) 850px-1200px
   4.X-large devices(digital screen) >1200px
